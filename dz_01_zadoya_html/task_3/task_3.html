What is HTML

Hypertext Murkup Language (HTML) is the standart markup landuage for creating web pages and web applications
Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages.HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.HTML provides a means to create structured documets by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items.HTML elements are delineared by tags, written using angle brackets.
